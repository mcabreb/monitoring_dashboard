## Dev Notes

**Project Structure:**
```
monitor_dashboard/
├── src/
│   └── monitor_dashboard/
│       ├── __init__.py             # Package init, version
│       ├── __main__.py             # Entry point: python -m monitor_dashboard
│       └── app.py                  # MonitorDashboardApp class
├── tests/
├── docs/
├── pyproject.toml
├── README.md
└── .gitignore
```

**pyproject.toml Configuration:**
- Build backend: hatchling
- Python version: >=3.10
- Use PEP 517/518 compliant structure
- Include optional [dev] dependencies group

**Minimal App Pattern:**
```python
from textual.app import App
from textual.widgets import Static
from textual.containers import Container

class MonitorDashboardApp(App):
    def compose(self):
        yield Container(Static("monitor_dashboard"), id="main")
```

**Coding Standards:**
- Use absolute imports within package
- Type hints on all public functions
- Google-style docstrings
- Never use print() - use logging module

### Testing

**Test Location:** tests/unit/

**Testing Standards:**
- Framework: pytest 8.0.0
