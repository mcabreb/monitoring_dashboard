## Story

**As a** developer,
**I want** a properly structured Python project with all dependencies configured,
**so that** I can start building the monitoring dashboard with a solid foundation.

## Acceptance Criteria

1. Project has `pyproject.toml` with metadata, dependencies (textual, psutil, pyyaml), and dev dependencies (pytest, black, ruff, mypy)
2. Source code is organized under `src/monitor_dashboard/` with `__init__.py` and `app.py`
3. Running `python -m monitor_dashboard` launches a minimal textual app displaying "monitor_dashboard" in a bordered container
4. Project includes `.gitignore` appropriate for Python projects
5. Basic `README.md` exists with project name and run instructions

## Dev Notes

**Project Structure:**
```
monitor_dashboard/
├── src/
│   └── monitor_dashboard/
│       ├── __init__.py             # Package init, version
│       ├── __main__.py             # Entry point: python -m monitor_dashboard
│       └── app.py                  # MonitorDashboardApp class
├── tests/
├── docs/
├── pyproject.toml
├── README.md
└── .gitignore
```

**pyproject.toml Configuration:**
- Build backend: hatchling
- Python version: >=3.10
- Use PEP 517/518 compliant structure
- Include optional [dev] dependencies group

**Minimal App Pattern:**
```python
from textual.app import App
from textual.widgets import Static
from textual.containers import Container

class MonitorDashboardApp(App):
    def compose(self):
        yield Container(Static("monitor_dashboard"), id="main")
```

**Coding Standards:**
- Use absolute imports within package
- Type hints on all public functions
- Google-style docstrings
- Never use print() - use logging module

### Testing

**Test Location:** tests/unit/

**Testing Standards:**
- Framework: pytest 8.0.0
- Run with: `pytest tests/unit/`
- No tests required for this story (infrastructure setup)
- Verify manually that app launches without errors

**Verification:**
```bash
# Install in dev mode
pip install -e ".[dev]"

# Verify app launches
python -m monitor_dashboard

# Verify code quality
black --check src/
ruff check src/
mypy src/
```
